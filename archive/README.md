# Archive

This folder keeps code and outputs that are no longer part of the active local pre-retrieval/retrieval workflow.

Archived here:

- old GraphDB / SPARQL-dependent retrieval and generation assets
- duplicate or superseded pre-retrieval helper scripts
- stale evaluation code from older retrieval methods
- legacy generated retrieval result files and demo question data

The active local workflow now lives under `src/pre_retrieval/` and writes fresh retrieval outputs under `data/retrieval_results/{representation}/`.
